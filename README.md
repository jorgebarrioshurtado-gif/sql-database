# IBM Employee Attrition Analysis (SQL Project)

## 📌 Introduction

This project was developed as part of a class assignment simulating a real-world business scenario. In this case, **IBM** has commissioned an analysis to better understand employee attrition within the company.

The objective is to determine whether attrition is a significant issue and to identify the key factors that contribute most to employees leaving the organization. Using a dataset on employee records, the project walks through the complete data workflow: data cleaning, database design, data population, and analytical querying.

By structuring the data into a relational database and running targeted SQL queries, the project aims to extract meaningful insights that could support data-driven decision-making around employee retention.

## 📁 Project Files

* **HR-Employee-Attrition.csv**
  Raw dataset sourced from Kaggle containing employee information prior to any processing.

* **ibm_data.csv**
  Cleaned and prepared dataset used for database population and analysis.

* **miniproject_cleaning.ipynb**
  Jupyter Notebook used for initial data exploration and preprocessing of the raw dataset.

* **mini_project.png**
  Visual representation of the database schema, including tables and relationships.

* **miniproject_schema.sql**
  SQL script defining the database structure (tables, fields, and relationships).

* **miniproject_populate_tables.sql**
  SQL script used to insert the cleaned data into the database tables.

* **miniproject_insights.sql**
  SQL queries designed to analyze the data and uncover insights related to employee attrition.

## 🧠 Project Overview

This project demonstrates an end-to-end data workflow in a SQL environment:

* Preparing and cleaning raw data
* Designing a relational database schema
* Populating structured tables
* Performing analytical queries to uncover trends and patterns

The final outcome is a set of insights that highlight potential drivers of employee attrition, showcasing how SQL can be used as a tool for business analysis.
